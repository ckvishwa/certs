# PRODUCT

## What the product is

An adaptive exam-preparation operating system. It continuously determines what
the learner knows, doesn't know, has forgotten, and repeatedly confuses — then
prescribes the next 30–60 minutes of highest-value study.

**It is not** a course website, a static LMS, or a video-plus-progress-bar app.

## Core promise

> "Open the app. It tells you exactly what to study next."

## The product loop

`ASSESS → PLAN → LEARN → RECALL → APPLY → TEST → ANALYZE → RESCHEDULE → REPEAT`

Every learner action improves the learner model. Every dashboard answers *"what
should I do next?"*. Every mistake answers *"why did I get this wrong?"*.

## Non-negotiable principles

- Learning is not content consumption.
- Completion is not mastery. One correct answer is not knowledge.
- Confidence matters; mistakes are data; forgetting is expected.
- Reviews must be scheduled before knowledge disappears.
- Weak **prerequisites** must be repaired (knowledge graph).
- The **official syllabus** is the source of truth. Historical practice papers
  inform practice but never redefine the curriculum.
- Optimize for exam success, not screen time. No manipulative streak mechanics.

## Initial certifications

- **Cisco CCNA 200-301 v1.1** (active). v2.0 structure modelled for a future
  transition mode (v2.0 testing begins 2027-02-03). v1.1 and v2.0 objectives are
  never mixed invisibly.
- **CompTIA Security+ SY0-701** (five official domains).

Syllabus hierarchy: `Certification → Version → Domain → Objective → Sub-objective
→ Concept → Skill → Question → Flashcard → Practical scenario`. Never flattened.

## MVP scope (Phase 1)

Auth · certification/version selection · official syllabus hierarchy · dashboard ·
knowledge map · flashcards · spaced repetition · quiz engine · error log · daily
study plan · basic readiness score · document/question-paper upload + extraction
review · responsive UI.

Deferred to Phase 2/3: adaptive diagnostic, confidence system, teach-back/blurt,
subnetting dojo, CLI labs, boss fights, mock exams, admin. See `ROADMAP.md`.
